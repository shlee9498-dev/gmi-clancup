# GmI Casino Bot v1.0

GmI 클랜 활동 → 코인 적립 → 상점 보상 + 잭팟 풀 PvP 시스템.

## 시즌1 (2026-05-20 ~ 2026-06-10)

### 코인 룰
| 게임 | 1등 | +1000딜 | +1000딜+8킬 |
|---|---|---|---|
| 일겜 (내부) | 4 | 5 | 6 |
| 경쟁전 (외부) | 1 | 2 | 3 |

경쟁전은 결과 캡쳐 첨부 필수, 자동 지급, 사후 검증 시 회수 가능.

### 잭팟 풀
- 운영 시간: **22:00 ~ 02:00 KST** (시간 외 차단)
- 1인 베팅 한도: **본인 보유 코인의 20%**
- Rake: **풀 전액의 10% 자동 소각**
- 최소 참가자: 2명 (미달 시 환불)
- 생성자 본인 참가 불가 (자전거래 방지)

### 상점
- 시즌1 4종 시드 상품
- 🍗 치킨 기프티콘은 **시즌 1회 제한**

## 접근 제어 (역할 기반)

봇의 모든 사용자 명령은 **`CLAN_ROLE_IDS`에 등록된 디스코드 역할 보유자**만 사용할 수 있습니다.
미보유 시 `❌ G/m/I 클랜원 역할 보유자만 사용 가능한 명령입니다.` 메시지가 출력되고 명령이 차단됩니다.

**Fail-closed 정책**: `CLAN_ROLE_IDS`가 비어 있으면 운영진(`ADMIN_IDS`) 외에는 누구도 봇을 사용할 수 없습니다. 미설정 시 봇이 사실상 잠긴 상태로 시작하니, **배포 직후 역할 ID 입력은 필수**입니다.

### 역할 ID 가져오는 방법 (모바일)
1. 디스코드 앱 → 서버 설정 → 역할 → G/m/I 역할 우측 점 3개
2. **"ID 복사하기"** (개발자 모드 켜져 있어야 함: 설정 → 고급 → 개발자 모드)
3. `CLAN_ROLE_IDS=g_id,m_id,i_id` 형식으로 Railway Variables에 입력
