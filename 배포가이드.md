## 📱 모바일에서 푸시하는 3가지 방법

### 방법 A: GitHub 웹 (가장 간단, 5분)

1. **모바일 브라우저로 github.com 접속 → 로그인**
2. `shlee9498-dev/gmi-clancup` 레포 진입
3. 우측 상단 **Add file → Upload files** 탭
4. zip 풀어서 받은 4개 파일 한꺼번에 드래그/선택:
   - `index.html`
   - `casino.html`
   - `casino-poster.png`
   - (선택) md 파일 2개도 같이 올려도 OK
5. 화면 아래로 스크롤 → **Commit changes** 메시지 입력:
   ```
   카지노 시즌1 페이지 추가 + 메인 통합 포털 개편
   ```
6. **Commit changes 버튼 클릭**
7. **1~3분 후 사이트 자동 반영** (GitHub Pages는 자동 빌드)
8. https://shlee9498-dev.github.io/gmi-clancup/ 새로고침해서 확인

⚠️ **주의:** `index.html`을 업로드하면 "이미 존재합니다, 덮어쓸까요?" 물어봄. **덮어쓰기 OK**.

---

### 방법 B: GitHub 모바일 앱 (편하지만 PNG 업로드 제한)

1. GitHub 앱 설치 (안드 플레이스토어)
2. `shlee9498-dev/gmi-clancup` 열기
3. 파일별로 편집:
   - `index.html` 우상단 ✏️ → 내용 전부 지우고 새 파일 내용 붙여넣기 → Commit
   - `casino.html` → Create new file → 파일명 입력 → 내용 붙여넣기 → Commit
4. PNG는 앱에서 업로드 불가 → **이 경우 PNG만 웹에서 별도 업로드** (위 방법 A 사용)

## 🌐 도메인 연결 (선택, 추후 작업)

`gmi-clan.com` 같은 도메인 산 뒤:

1. 가비아/Cloudflare DNS 관리에서:
   ```
   A 레코드 4개:
     185.199.108.153
     185.199.109.153
     185.199.110.153
     185.199.111.153
   CNAME (www): shlee9498-dev.github.io
   ```
2. 레포 Settings → Pages → Custom domain → `gmi-clan.com` 입력
3. CNAME 파일 자동 생성됨
4. "Enforce HTTPS" 체크
5. 24시간 이내 SSL 인증서 자동 발급
6. 완료 후 https://gmi-clan.com 으로 접속

## 🐛 자주 발생하는 문제

| 증상 | 원인 | 해결 |
|---|---|---|
| 배너 이미지가 안 보임 | PNG 업로드 누락 | `casino-poster.png` 별도 업로드 확인 |
| 메인은 됐는데 `/casino.html` 404 | 파일 업로드 실패 | 다시 업로드 |
| 화면 안 바뀜 | 브라우저 캐시 | 강제 새로고침 (모바일: 주소창 길게 누르고 "캐시 삭제 후 새로고침") |
| 카지노 카드만 깨짐 | `casino-poster.png` 경로 오타 | 파일명이 정확히 `casino-poster.png`인지 확인 (소문자) |
| GitHub Pages 빌드 실패 | HTML 구문 오류 | 레포 Actions 탭에서 에러 메시지 확인 |
